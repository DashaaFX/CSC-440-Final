# Campus Work Order System

Our final project will be a website allowing students to submit work orders for any matters requiring attention on campus. Students may submit requests, and faculity and admins may modify requests.

## Stack

### Backend
For backend we will use the following technologies:
- Python + Flask for API and serving HTML
- MySQL + SQLAlchemy for database record storing and access

### Frontend
- Plain HTML for web pages
- CSS framework TBD or none
